# RAG Document Chat System

A production-ready Retrieval-Augmented Generation (RAG) chatbot. Upload PDF, DOCX, PPTX, XLSX, or TXT documents and chat with them intelligently.

## Features
- 📄 Upload & process PDF / DOCX / PPTX / XLSX / TXT documents
- 💬 Chat with your documents using RAG (Retrieval-Augmented Generation)
- 🛡️ Prompt injection & out-of-scope guardrails
- 🔍 Semantic search via Qdrant vector database
- 📊 Chat history stored in PostgreSQL
- ⚡ Powered by LlamaIndex + OpenRouter + Cohere Reranker
